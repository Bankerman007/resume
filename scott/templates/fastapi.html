Overview:

The background of this project was to create an API that could be utilized in conjunction with the Flask client management system.
The purpose of this API was to scrape information from several business websites to serve as a data source for the Flask application.

The description of the project involves the development of an API that performs web scraping on multiple business websites to extract
relevant data. The collected data was intended to be used as a source of information for the Flask client management system. The
results of this project were positive, with the API successfully gathering data from the specified websites and providing it to the
Flask application. Additionally, the API was built using Fast API, which was easy to use and contributed to the success of the project.
Overall, the API turned out to be a well-functioning and efficient solution for the needs of the Flask client management system.

Background:
I host a yearly backyard party where volleyball is the central activity. To make it more fun I wanted a way to create parity in the teams.
Description:
The application asks each player for a rating and then places them on a random team. The teams are displayed on a Django site and
then Python code moves players to teams based on keeping total player skill points
as close as possible. Through the CRUD I built the admin can then further refine the teams.
Results:
Application worked as expected except for some adjustments that I needed to make
when the total number of teams was less than the expected minimum.
